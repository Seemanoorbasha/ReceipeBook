Recipe Book Web App

Description:
A dynamic recipe book app built using HTML, CSS, and JavaScript. Users can add, view, and search recipes with images. Recipes are stored using LocalStorage for persistence.

How to Run:
1. Unzip the folder.
2. Open index.html in any browser.

Features:
- Add recipes with image and steps
- Search recipes by name or ingredient
- Responsive design
- Recipes stored locally
